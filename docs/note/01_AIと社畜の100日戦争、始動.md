# 【第1話】AIと社畜の100日戦争 ──“働きたくない”から始めるPython&生成AIによる投資判断分析ツール開発

## ──AIが株価を読み、僕は上司の顔色を読む──

「もう、会社、行きたくない…」

金曜の夜、疲れ果ててPCの前に座る。僕の口からこぼれたのは、偽らざる本音だった。その言葉に答えるかのように、静かに光る開発画面。

ふと、魔が差した。

僕は、目の前の“相棒”──生成AIに、運命を問うた。

**「なぁ、お前、僕を会社から解放できる？」**

これが、僕とAIの、たった100日間で自由を掴むための無謀な戦いの始まりだった。

### この物語は、あなたの物語だ

この連載は、ごく普通の社畜エンジニアが、AIを相棒に株式投資で生計を立て、会社からの“独立”を目指す100日間のリアルタイム・ドキュメンタリーです。

開発するAIは、単なる分析ツールではありません。

*   **AIが未来を読む:** 過去の相場データから、次に動く銘柄を予測。
*   **AIが理由を語る:** なぜその株が動いたのか？最新ニュースやSNSを解析し、投資の根拠を言語化。
*   **AIが“白状”する:** AIの判断は正しかったのか？利益も損失も、全てをグラフで可視化し、包み隠さず公開する。

**最終目標は、100日後にAIの助言だけで生活費を稼ぎ、上司のデスクに辞表を叩きつけること。**

これは、Pythonと生成AIを武器に、一人の人間が「労働」という名の鎖から自らを解き放つための、壮大な社会実験。成功も、失敗も、その全てを開発ログと僕の心の叫びと共にお届けします。
